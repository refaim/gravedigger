# План: CP866 шрифт в EXE для exit screen

## Контекст

- Exit screen — VGA text mode 80x25, символы рисуются аппаратным шрифтом VGA (по умолчанию CP437)
- Чтобы кириллица отображалась, перед показом exit screen нужно загрузить CP866 шрифт через `INT 10h, AX=1110h`
- В decompressed EXE по code offset `0x679` лежит `B8 03 00 CD 10` (`mov ax,3; int 10h` — установка текстового режима)
- Заменяем эти 5 байт на `9A 00 00 xx xx` (far call на stub), stub ставит режим + грузит шрифт + retf
- Шрифт берётся из XBIN (`xb.font`) при repack — пользователь может подставить свой через Moebius
- Шрифт и stub (~36 байт) дописываются в конец EXE, paragraph-aligned
- Нужны 2 новые записи в таблице релокаций MZ (место есть — 3 свободных слота)
- Если в XBIN есть шрифт — вшиваем в EXE; если нет — не патчим
